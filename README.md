> 一个适用于HBuilderX的PDF查看扩展


## `这是IDE插件，不是uni-app开发用插件！！！`

**本插件旨在支持直接在HBuilderX内查看PDF文件，安装后无需任何配置**

**正常点击打开需要查看的PDF文件即可**
